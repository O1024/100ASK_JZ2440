# JZ2440 SDRAM 实验

本实验旨在学习如何初始化 S3C2440 的内存控制器，以驱动开发板上的外部 SDRAM (EM63A165)。

## 实验目标
1. **时钟配置**：确保 HCLK 处于合适频率（通常为 100MHz），为内存控制器提供时钟。
2. **内存控制器初始化**：配置 `BWSCON` 及 `BANKCON6` 等寄存器，匹配 SDRAM 的硬件特性（如列地址线数量、CAS Latency 等）。
3. **代码重定位**：将存储在 Nand Flash 或内部 SRAM 中的程序镜像复制到 SDRAM (0x30000000) 中运行。
4. **验证**：通过在 SDRAM 中执行代码或读写测试，验证初始化是否成功。

## 硬件信息
- **SDRAM 型号**：EM63A165 (32MB)
- **连接 Bank**：BANK6
- **起始地址**：0x30000000
- **位宽**：32-bit (由两片 16-bit 芯片并联)
